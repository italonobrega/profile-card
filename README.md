# 🪪 Profile Card

Card de perfil desenvolvido com HTML, CSS e JavaScript puro.

**[👉 Clique aqui para acessar o site](https://italonobrega.github.io/profile-card/)**

## ✨ Funcionalidades

- Foto de perfil com recorte circular
- Seção de skills com ícones
- Repositórios e seguidores do GitHub atualizados em tempo real via API
- Botões de contato: WhatsApp, GitHub e Email
- Layout responsivo para mobile
